# Pokemon Məlumat Analizi | Python

## Layihəyə Ümumi Baxış
Pokemon məlumat dəsti Python vasitəsilə təhlil edilərək
fərqli tip, güc və xüsusiyyətlər arasındakı əlaqələr
kəşf edilib. Layihə EDA (Kəşfiyyat Məlumat Analizi)
bacarıqlarını nümayiş etdirmək məqsədilə hazırlanıb.

## İstifadə Edilən Alətlər və Kitabxanalar
- Python 3.x
- Pandas (məlumat təmizlənməsi və analizi)
- Matplotlib / Seaborn (vizuallaşdırma)
- NumPy (statistik hesablamalar)
- Jupyter Notebook

## Analiz Mərhələləri
- Məlumatların yüklənməsi və ilkin yoxlanması
- Boş və səhv dəyərlərin təmizlənməsi
- Deskriptiv statistika
- Tip üzrə güc müqayisəsi
- Korrelyasiya analizi
- Vizual hekayəçilik

## Əsas Tapıntılar
- Ən güclü Pokemon tipi: XX
- Hücum və müdafiə arasında XX korrelyasiya aşkar edilib
- Legendary Pokemon-ların ortalama statistikası
  adi Pokemon-lardan XX% yüksəkdir

## Nümayiş Edilən Bacarıqlar
- Pandas ilə məlumat manipulyasiyası
- Matplotlib / Seaborn ilə vizuallaşdırma
- Statistik düşüncə və EDA
- Jupyter Notebook ilə təmiz kod yazımı
